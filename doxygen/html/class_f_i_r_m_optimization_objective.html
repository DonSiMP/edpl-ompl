Detailed Description

The cost to execute an edge i.e. going from some node A to some node B is defined as the FIRM optimization objective. The cost is a measure of the uncertainty in the system (e.g. trace(covariance) for all intermediate states on the edge). FIRM gives us paths that have the least uncertainty.
